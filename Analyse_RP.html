Vergleich der Mittelwerte der Interessenssteigerung

Berechnung der Summen

Fr den Vergleich mssen wir erst die Summen der Items berechnen und dann die Summen des Interesses vor Betrachtung des Stimulus von denen nach der Betrachtung des Stimulus subtrahieren.

daten <-
daten %>% 
  select(!c(SERIAL:STARTED, MAILSENT, LASTDATA, MISSING, MISSREL, MAXPAGE, 
            R001_CP, TIME001:TIME004, TIME006:TIME011))
daten <- daten %>% 
   row_sums(daten[21:24], n = 4, var = "sum_ibefore") %>% 
   row_sums(daten[25:28], n = 4, var = "sum_iafter")

daten <- daten[complete.cases(daten[c("sum_ibefore", "sum_iafter")]), ]

daten <- daten %>% 
  mutate(sum_diff = sum_iafter - sum_ibefore)

Berechnung und Visualisierung der Mittelwerte

Nun knnen wir einfach die Mittelwerte der Differenzen bilden und visualisieren.

daten <- daten %>%
  group_by(R001) %>% 
  mutate(mean_diff = mean(sum_diff))

ggbetweenstats(
  daten,
  x = R001,
  y = sum_diff,
  results.subtitle = FALSE
)

Jetzt sehen wir die Unterschiede in der Interessenssteigerung fr jede Gruppe. Die Unterschiede fallen uerst gering aus, nichtsdestotrotz ist ein leichter Unterschied zu erkennen, der sich mit unserer Hypothese deckt, nmlich fllt die Interessenssteigerung beim animierten Graph hher aus, als beim statischen und die Steigerung beim interaktiven wiederum hher als beim animierten. Bemerkenswert ist auerdem, dass es bei Gruppe 3 nur einen Fall gibt, bei dem sich eine Interessensminderung abzeichnet, whrend es bei den anderen beiden Gruppen sowohl mehr, als auch extremere Flle dieser Art gibt.

Berechnung des Modells

Nun berechnen wir das Modell, als die Interessenssteigerung in Abhngigkeit von der Versuchsgruppe:

daten1 <- daten %>% 
  mutate(R001 = factor(R001))

lm1 <- stan_glm(sum_diff ~ R001, 
                data = daten1, 
                refresh = 0)

Prfen auf Signifikanz

Interpretation der Modellparameter

Nun sehen wir uns die Koeffizienten des Modells an:

parameters(lm1)
parameters(lm1) %>% plot(show_intercept = TRUE)

Der Gruppenunterschied wird auf 0.31, 0.22 geschtzt; das ist der Punktschtzer der UV R001. Die Grenzen eines 95%-CI fr die UV liegen bei 0.24 bzw. 0.35. Dieser Bereich schliet die 0 aus, das heit, dass auf jeden Fall ein Effekt vorliegt.

ROPE-Test

Testen der Hypothese mit dem ROPE-Verfahren:

rope(lm1)
rope(lm1) %>% plot()

Die Verteilungen liegen zu 4%, 32% und 17% im ROPE, das heit wir knnen keinen signifikanten Effekt feststellen.

Fazit

Fr den Effekt des Diagrammtyps ist keine klare inferenzstatistische Aussage hinsichtlich eines Indifferenzbereichs mglich: Es ist plauibel, laut dem Modell, dass es einen praktisch bedeutsamen Unterschied gibt, aber es ist auch plausibel, dass es keinen praktisch bedeutsamen Unterschied gibt.
